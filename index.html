About Me

My name is Éinín (yes, it's Irish; it means "little bird.") I'm a 23 year old aspiring front-end web developer who already knows HTML and CSS and is in the process of learning Javascript (among other things.) My "thing" is that I like to design websites for things that do not exist (see below.)

I started doing "basic web design" as a kid/young teen on Neopets and started "taking it seriously" last year. I've been getting up to date on my best practices (which were roughly 15 years out of date...) and working on making my code cleaner and more legible, as well as simply refining my skills into something workable on the modern internet.

I'm also learning about coding best practices in general-- learning to use Visual Studio Code efficiently (instead of typing everything line by line into the native Notepad app on Windows, yikes!) and practicing working with other useful tools such as Bootstrap and Sass.

My main goal is to prioritize accessibility and user experience in my designs, including mobile-friendly accessibility as I've recently learnt how many people view websites on their phones first and foremost.

Skills

Right now the only skill that I am entirely confident in my abilities in is HTML/CSS. I have been doing it for a very, very long time (I started before I was 10, so I have a decade of experience!) and I consider myself very good at it. I would describe myself as "extremely proficient."

Some other web skills I have include using Bootstrap to build webpages. I learnt it in the span of a few hours to build custom character profiles on a website called toyhou.se for my best friend (who is also my brother.) I have built multiple standalone webpages using Bootstrap since, using some custom colors and elements as well as some with overrides in an external stylesheet.

You can view the first webpage I built with Bootstrap here, which uses some custom styles but is mostly raw Bootstrap. It is a webpage for a fictional organization from the video game Overwatch. The second page I built in Bootstrap is similar. I chose to use Bootstrap in order to simulate the design of a corporate website, which I believe comes across more efficiently in the design first example but is more accurate in the second.

I am also learning to use Sass, which can go hand in hand with developing with Bootstrap; it's been a challenge to learn how to set up my development environment for more complicated projects, but I'm getting there. While I haven't yet had a project large enough to mentally justify using both Bootstrap and Sass at once, I highly suspect these skills will come into use in the near future, given my plans...!

I am learning to use Javascript. I am not good at it yet-- programming is something I've been struggling to get into for a long time. It's a difficult process, and a bit slow for me (which is crazy! I'm not used to having trouble picking things up!) but I think I'm making decent progress and I'm proud of my work.

roleplayer.info

Roleplayer.info is a project I'm working on to help roleplayers around the internet. Roleplaying is one of my favorite online hobbies and I was always looking for new places to find partners but constantly found myself wishing there were more information about the individual cultures and nuances of each website or app... so I'm doing it myself! Roleplayer.info is a website dedicated to helping people find places to roleplay online as well as teach them what to expect and how to best use them! I'm hoping to do cultural writeups on each in order to make it easier for new members to utilize them as effectively as possible.
